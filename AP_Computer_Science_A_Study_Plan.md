# AP Computer Science A - 12 Month Study Plan
**Target Exam Date:** May 2027  
**Start Date:** May 2026  
**Study Method:** Self-study with Khan Academy, Albert.io, and supplemental resources

---

## 📚 Required Resources

### Free Resources
- **Khan Academy** - AP Computer Science A course
- **CodingBat** - Java practice problems (codingbat.com/java)
- **Introduction to Programming Using Java** - Free online textbook
- **College Board** - Official AP CS A course description & practice tests
- **Albert.io** - Practice questions (free trial, then paid)

### Recommended Purchases
- **Barron's AP Computer Science A** (harder than actual exam - ideal for prep)
- **Flashcard app** - Anki (free) or Quizlet

### Tools Needed
- Computer with internet connection
- Java Development Kit (JDK) installed
- IDE: IntelliJ IDEA Community Edition or Eclipse (both free)

---

## 🎯 Exam Structure (2026+ Format)

**Total Time:** 3 hours  
**Section I:** Multiple Choice (40 questions, 90 min, 50% of score)  
**Section II:** Free Response (4 questions, 90 min, 50% of score)

**Content Breakdown:**
- Programming Fundamentals: 55%
- Object-Oriented Programming: 25%
- Data Structures: 15%
- Algorithm Analysis: 5%

## 📅 Month-by-Month Study Plan

### **Month 1-2: Java Fundamentals (May-June 2026)**

#### Week 1-2: Setup & Introduction
- [ ] Install JDK and IDE (IntelliJ recommended)

- [ ] Review AP Java Subset document
- [ ] Create flashcard deck for vocabulary

- [ ] Learn: String class basics

- [ ] CodingBat: String-1 problems

- [ ] Practice: Write 5 simple programs daily

#### Week 5-6: Control Structures

#### Week 7-8: Loops

- Review week's material

## 🎯 Key Success Strategies

1. **Code Daily** - Even 30 minutes of coding is better than none
2. **Use Flashcards** - Memorize syntax, methods, and concepts
3. **Practice FRQs** - Start practicing free response early (Month 6+)
4. **Learn from Mistakes** - Review every wrong answer thoroughly
5. **Time Yourself** - Practice under timed conditions
6. **Write Code by Hand** - The exam is on paper, not a computer
7. **Join Online Communities** - r/APStudents for support and tips
