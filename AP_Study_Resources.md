## 🎯 Recommended Study Strategy

### **Daily Routine**

**AP Computer Science A (1.5-2 hrs/day)**
- 30 min: Khan Academy lessons
- 45 min: CodingBat practice
- 15 min: Flashcard review
- 15 min: Code a small project/problem

**AP Calculus BC (2-2.5 hrs/day)**
- 45 min: Khan Academy or Professor Leonard videos
- 60 min: Practice problems (textbook, Albert.io, or Paul's Notes)
- 15 min: Flashcard review (formulas, theorems)
- 15 min: Calculator practice (once you have one)

### **Weekend Deep Dives**

**Saturday (6-8 hours total)**
- 3-4 hours: CS A (longer coding projects, practice tests)
- 3-4 hours: Calc BC (problem sets, FRQ practice)

**Sunday (4-5 hours total)**
- 2 hours: CS A review
- 2 hours: Calc BC review
- 1 hour: Plan next week, review flashcards

## 📅 Resource Usage Timeline

### **Months 1-3: Foundation Building**
- **CS A:** Khan Academy + CodingBat daily
- **Calc BC:** Khan Academy + Professor Leonard + Paul's Notes

### **Months 4-6: Skill Development**
- **CS A:** Continue above + start Albert.io
- **Calc BC:** Continue above + start practice problems from review books

### **Months 7-9: Advanced Topics**
- **CS A:** Advanced OOP, algorithms, recursion
- **Calc BC:** Series, parametric, polar (BC-specific content)

### **Months 10-12: Exam Prep**
- **CS A:** Practice tests, past FRQs, weak area review
- **Calc BC:** Practice tests, 50+ past FRQs, formula memorization

## 🚨 Critical Reminders

### **AP Computer Science A**
- [ ] Install Java JDK and IDE by Week 1
- [ ] Start coding DAILY from Day 1
- [ ] Practice writing code BY HAND (exam is on paper)
- [ ] Do at least 20 past FRQs before exam
- [ ] Join r/APStudents by Month 3

### **AP Calculus BC**
- [ ] Buy graphing calculator by Month 3
- [ ] Master calculator functions by Month 6
- [ ] Memorize common Taylor series
- [ ] Do at least 50 past FRQs before exam
- [ ] Create formula sheet (even though one is provided)
- [ ] Practice showing work clearly (FRQs require justification)

### **Both Exams**
- [ ] Register for exams by early March 2027
- [ ] Find a local school that accepts self-study students
- [ ] Take 5+ practice tests for each exam
- [ ] Review scoring guidelines for FRQs
- [ ] Get adequate sleep before exam day
