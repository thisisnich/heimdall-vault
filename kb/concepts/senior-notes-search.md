# Senior Notes Search

- # Senior Notes Search & Integration Prompt

> **Quick use:** Paste this to an AI agent when you need help finding relevant senior notes.

---

## THE PROMPT

```
I'm working on [TOPIC] for [COURSE CODE]. 

Search the Senior Notes library at `Senior Notes/[COURSE FOLDER]/` for relevant materials.

STEPS:
1. Search for files matching the topic keywords
2. List the top 5 most relevant files with brief descriptions
3. For the most relevant file, show me:
   - Key concepts covered
   - How it differs from my current understanding
   - Specific sections I should focus on
   - Any warnings (outdated info, different tools, etc.)

4. Suggest how to integrate this into my own notes with proper attribution

IMPORTANT:
- Prioritize files with clear explanations and examples
- Flag any content that seems outdated (check dates, tool versions)
- Don't just copy — help me understand and synthesize
- Remind me to verify with official course materials

COURSE FOLDER MAPPING:
- EGE301 → Senior Notes/EGE301 CWS/
- EGE320 → Senior Notes/EGE320 ESD/
- EGE321 → Senior Notes/EGE321 WCN/
- EGE322 → Senior Notes/EGE322 ISP/
```
