## 📝 Question Format

Each question must have:
- **question**: The question text (string)
- **options**: Array of 4 answer choices (strings)
- **correct**: Index of correct answer (0-3)
- **explanation**: Why the answer is correct (string)

## 💡 Tips for Writing Good Quizzes

### **Questions:**
- Be clear and concise
- Avoid ambiguous wording
- Test understanding, not memorization
- Use realistic scenarios when possible

### **Options:**
- Make all options plausible
- Keep similar length
- Avoid "all of the above" or "none of the above"
- Mix up position of correct answer

### **Explanations:**
- Explain WHY the answer is correct
- Reference key concepts
- Point out common mistakes
- Keep it concise but informative

## 🎨 Organizing Quizzes

### **Multiple Topics:**
You can have multiple topics in one quiz deck:

```javascript
quizzes: [
    { title: "Chapter 1", questions: [...] },
    { title: "Chapter 2", questions: [...] },
    { title: "Final Review", questions: [...] }
]
```

### **Question Difficulty:**
Consider marking difficulty in question text:
- "⚠️ CRITICAL:" for must-know questions
- "🔥 HARD:" for challenging questions
- "⭐ BONUS:" for extra credit
