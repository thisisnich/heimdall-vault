### Key Formatting Rules
1. **Code block**: Always wrap ladder diagrams in triple backticks (```)
2. **Rung labels**: Start each rung with `Rung N:` where N is the rung number
3. **Power rail**: Start each line with `|--` to represent the left power rail
4. **Contacts**: Use `[ADDRESS]` for contacts
5. **Coils**: Use `(ADDRESS)` for coils at the end
6. **Branch markers**: Use `+` at branch points, align vertically with `|`
7. **Spacing**: Use hyphens `----` to represent wires between elements
8. **Right rail**: End with `--|` to represent the right power rail (optional)
9. **Counter blocks**: Use two lines for counters - top line for increment (CT), bottom line for preset (U)

### Alignment Guidelines
- Keep branch points aligned vertically using `|`
- Use consistent spacing between elements
- Longer rungs may need more hyphens for visual alignment
- Counter blocks use two lines - align the preset (U) line under the increment (CT) line
