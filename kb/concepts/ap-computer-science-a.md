## 🖥️ AP Computer Science A Resources

### **Primary Learning Platforms (FREE)**

#### 1. **Khan Academy** ⭐ ESSENTIAL
- **Link:** https://www.khanacademy.org/computing/ap-computer-science-principles
- **What:** Complete structured AP CS A course
- **Why:** Best free comprehensive course, structured learning path
- **Use for:** Primary learning resource, daily lessons

#### 2. **CodingBat** ⭐ ESSENTIAL
- **Link:** https://codingbat.com/java
- **What:** Interactive Java practice problems
- **Why:** Immediate feedback, hundreds of problems, organized by difficulty
- **Use for:** Daily coding practice (30-60 min/day)

#### 3. **College Board Official Resources** ⭐ ESSENTIAL
- **Past FRQs:** https://apcentral.collegeboard.org/courses/ap-computer-science-a/exam/past-exam-questions
- **Course Description:** https://apcentral.collegeboard.org/courses/ap-computer-science-a
- **Java Subset:** https://apcentral.collegeboard.org/media/pdf/ap-computer-science-a-java-subset.pdf
- **Why:** Official exam questions, understand exact requirements
- **Use for:** Practice FRQs (start Month 6), understand exam format

## 📚 Required Resources

### Free Resources
- **Khan Academy** - AP Computer Science A course
- **CodingBat** - Java practice problems (codingbat.com/java)
- **Introduction to Programming Using Java** - Free online textbook
- **College Board** - Official AP CS A course description & practice tests
- **Albert.io** - Practice questions (free trial, then paid)

### Recommended Purchases
- **Barron's AP Computer Science A** (harder than actual exam - ideal for prep)
- **Flashcard app** - Anki (free) or Quizlet

### Tools Needed
- Computer with internet connection
- Java Development Kit (JDK) installed
- IDE: IntelliJ IDEA Community Edition or Eclipse (both free)

---

## 🎯 Exam Structure (2026+ Format)

**Total Time:** 3 hours  
**Section I:** Multiple Choice (40 questions, 90 min, 50% of score)  
**Section II:** Free Response (4 questions, 90 min, 50% of score)

**Content Breakdown:**
- Programming Fundamentals: 55%
- Object-Oriented Programming: 25%
- Data Structures: 15%
- Algorithm Analysis: 5%
