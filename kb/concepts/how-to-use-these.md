# How To Use These

- 🎯 How to Use These Notes

### ✅ DO:
- **Cross-reference concepts** when you're stuck on a topic
- **Find alternative explanations** if lecture notes are unclear
- **Discover exam patterns** and common question types
- **Check formulas and definitions** for accuracy
- **Look for practical examples** and worked solutions
- **Use as study guide templates** for organizing your own notes

### ❌ DON'T:
- Copy answers verbatim (academic integrity)
- Assume content is 100% current (syllabi change)
- Skip your own lectures/labs to rely on these
- Use outdated code examples without verification
- Ignore differences in instructor teaching styles

---

## 🔍 Quick Search Strategy

When you need help with a topic:

1. **Search your own notes first** using Obsidian search
2. **If unclear, search Senior Notes** for the same topic
3. **Compare approaches** — what's different? What's similar?
4. **Synthesize** — combine the best of both into your understanding
