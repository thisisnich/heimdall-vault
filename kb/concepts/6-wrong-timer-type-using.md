# 6 Wrong Timer Type Using

- 6. **Wrong timer type**: Using TMY (1s) when you need TMR (10ms) for precision
7. **SV calculation errors**: Forgetting that SV is counts, not time
8. **Counter not reset**: Counters latch ON when complete and need explicit reset
9. **Duplicate outputs**: Using the same Y address in multiple rungs
10. **Address confusion**: Mixing up timer notation (TMxz) with timer contact (Tz)
