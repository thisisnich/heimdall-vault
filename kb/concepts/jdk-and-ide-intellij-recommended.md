# Jdk And Ide Intellij Recommended

- JDK and IDE (IntelliJ recommended)
- [ ] Read AP CS A Course Description (pages 4-10, 17)
- [ ] Review AP Java Subset document
- [ ] Create flashcard deck for vocabulary
- [ ] Khan Academy: Complete "Intro to Java" unit
- [ ] CodingBat: Warmup-1 section (all problems)

#### Week 3-4: Variables, Data Types, Operators
- [ ] Khan Academy: Variables and data types
- [ ] Learn: primitive types (int, double, boolean, char)
- [ ] Learn: String class basics
- [ ] Learn: arithmetic, relational, logical operators
- [ ] CodingBat: String-1 problems
- [ ] Albert.io: "Program Basics" section (20 questions)
- [ ] Practice: Write 5 simple programs daily

#### Week 5-6: Control Structures
- [ ] Khan Academy: If statements and boolean logic
- [ ] Learn: if, if-else, nested if, switch statements
- [ ] Learn: compound boolean expressions
- [ ] Learn: De Morgan's Laws
- [ ] CodingBat: Logic-1 and Logic-2 sections
- [ ] Albert.io: "Control Structures" (30 questions)

#### Week 7-8: Loops
- [ ] Khan Academy: Loops unit
- [ ] Learn: while loops, for loops, nested loops
- [ ] Learn: loop patterns (counting, accumulating, searching)
- [ ] CodingBat: Warmup-2, String-2, Array-1
- [ ] Practice: 10 loop problems daily
- [ ] Mini-project: Create a simple game (guess the number)
