# Tutorial Qa Summary Tutorial

- Tutorial Q&A Summary

### Tutorial 1 Key Points
- Project = temporary, unique, defined start/end
- Custom house = project; Standard house = business process
- Four phases: Initiation, Planning, Execution, Closure

### Tutorial 2 Key Points
- Benefits of initiation: clear direction, efficient allocation
- 5 authorization reasons: market, organizational need, customer request, new tech, compliance
- SWOT: analyze feasibility and allocate resources
- 3 steps: identify sponsor, appoint PM, identify core team
- Flexibility Matrix: prioritize scope/time/cost

### Tutorial 3 Key Points
- WBS: structured way to organize work, completeness is key
- Top-down: 4-7 major components, then break down
- Effort → Duration → Calendar Time
- Execution/monitoring: monitor, manage, track, communicate
- Closure: announce, review, debrief, reward, paperwork
