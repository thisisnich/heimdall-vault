## Basic Concepts

### What is Ladder Logic?
Ladder logic is a graphical programming language used to program PLCs (Programmable Logic Controllers). It mimics electrical relay logic diagrams, making it intuitive for engineers familiar with electrical circuits.

**This guide is for Panasonic FP PLCs using FPWIN GR7S software.**

### Key Components
- **Rungs**: Horizontal lines that represent parallel circuits (like rungs on a ladder)
- **Contacts**: Input conditions (like switches, sensors) - normally open or normally closed
- **Coils**: Outputs (relays, motors, lights)
- **Timers**: Time-delay elements
- **Counters**: Count events or pulses
- **Branches**: Parallel logic paths within a rung

## Symbol Conventions

### Input Contacts
| Symbol | Meaning | Description |
|--------|---------|-------------|
| `[X###]` | Normally Open Contact | Closes when input is ON |
| `[/X###]` | Normally Closed Contact | Opens when input is ON |
