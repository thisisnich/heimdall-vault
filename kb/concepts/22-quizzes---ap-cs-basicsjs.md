## 🚀 How to Use

1. **Right-click** on the HTML file in Obsidian's file explorer
2. Select **"Open in default app"** or **"Reveal in system explorer"**
3. Double-click the HTML file to open in your browser

## 📖 Universal Flashcards Guide

### **How It Works**
1. Open `universal-flashcards.html`
2. Select a deck from the dropdown
3. Choose a filter (All Cards, Critical Only, or By Chapter)
4. Study with keyboard shortcuts or mouse clicks

### **Features**
- ✅ Click card to flip between question and answer
- ✅ Navigate with Previous/Next buttons
- ✅ Filter by All, Critical, or Chapter
- ✅ Shuffle deck for random order
- ✅ Track progress and studied cards
- ✅ Beautiful Notion-inspired design

### **Keyboard Shortcuts**
- `←` **Left Arrow** - Previous card
- `→` **Right Arrow** - Next card
- `Space` or `Enter` - Flip card
