### Question 8
What is the value of variable `iResult` after executing the following program?

```c
int Add(int iNum1, int iNum2)
{
    return (iNum1 + iNum2);
}

int main(void)
{
    int iVar1, iVar2, iResult;
    iVar1 = 41;
    iVar2 = 43;
    iResult = Add(iVar1, iVar2);
}
```

**Options:**
- A. 41
- B. 43
- C. 84
- D. 48

**Answer:** C (84 - 41 + 43 = 84)
