## 🎯 Key Concepts

### PLC (Programmable Logic Controller)

**Components:**
- Power Supply
- CPU (e.g., FP2-C1)
- Network Module
- Input/Output Module (e.g., XY64D2T)

**I/O Addressing:**
- **Inputs (X):** Switches, sensors, pushbuttons
- **Outputs (Y):** Lamps, buzzers, motors, conveyors

**Features:**
- Rugged, noise-immune equipment
- Modular plug-in construction
- Standard I/O connections
- Easy programming and reprogramming
- Network communication capable

### Ladder Diagram
- Like a wiring diagram
- Each rung = logical circuit
- Read left to right, top to bottom
- Visual programming language for PLCs

### PLC Scan Cycle
1. **Input Scan:** Check all input devices → update input memory
2. **Program Execution:** Execute user program → update output memory
3. **Output Scan:** Transfer output memory → physical outputs
