# Ontmatter Replace The Fill In Stubs

- ONTMATTER
   Replace the FILL_IN stubs with real values based on the content:
   - tags: relevant tags (course code, topic keywords)
   - course: e.g. "EGE322 IoT System Project"
   - topic: short description of what this note covers

b) STRUCTURE THE CONTENT
   - Add a clear H1 heading if missing (use the document title verbatim)
   - Add section headers using existing section titles already present in the text — do NOT invent new headings
   - Do NOT reword, paraphrase, shorten, or summarise any sentence
   - Do NOT add explanatory text, notes, or commentary that wasn't in the original
   - Preserve all original content exactly as written

c) ADD CROSS-LINKS
   Based on the content, add wikilinks to related notes already in the vault.
   Use [[filename|display text]] format.
   Add a "Related:" line near the top, e.g.:
   > 📚 **Related:** [[ExistingNote|Label]] | [[AnotherNote|Label]]

d) CONFIRM PLACEMENT
   Tell me which course folder this file should go into, and ask me to confirm before moving it.
   If it's obvious (e.g. an EGE322 lecture slide), just tell me your recommendation.
