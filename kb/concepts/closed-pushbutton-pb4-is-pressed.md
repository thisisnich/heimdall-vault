# Closed Pushbutton Pb4 Is Pressed

- closed pushbutton (PB4) is pressed.
- 3)  Write a PLC program to turn an LED (Y122) after 5 seconds, when a normally-open pushbutton
(PB5) is pressed. The LED turns off when another normally-open pushbutton (PB6) is pressed.
- 4)  Write a PLC program to turn an LED (Y122) when a normally-open pushbutton (PB5) is pressed.
The LED turns off after 3 seconds, when another normally-open pushbutton (PB6) is pressed.
