# It Works 1 Open Universal Quizhtml

- It Works**
1. Open `universal-quiz.html`
2. Select a quiz from the first dropdown
3. Select a topic from the second dropdown
4. Click "Start Quiz"
5. Answer questions and get instant feedback

### **Features**
- ✅ Multiple choice questions with instant feedback
- ✅ Detailed explanations for each answer
- ✅ Score tracking and timer
- ✅ Results summary with grade (A-F)
- ✅ Clean Notion-style interface

### **Keyboard Shortcuts**
- `1-4` - Select answer A, B, C, or D
- `Enter` - Go to next question (after answering)
