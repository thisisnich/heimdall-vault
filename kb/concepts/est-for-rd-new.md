### Network Diagram (PERT/CPM)

A **network diagram** (also called **activity-on-node** diagram) visually represents:
- **Activities** as nodes/boxes
- **Dependencies** as arrows
- **Sequence** of tasks

**Types of Dependencies:**
- **Finish-to-Start (FS)** — Task A must finish before Task B starts (most common)
- **Start-to-Start (SS)** — Task A must start before Task B can start
- **Finish-to-Finish (FF)** — Task A must finish before Task B can finish
- **Start-to-Finish (SF)** — Task A must start before Task B can finish (rare)
