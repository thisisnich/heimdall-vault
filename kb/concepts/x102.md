# X102

- -|
|           |
|     /X102 |
|----[ ]----+
```

**Operation:** Press PB1 (X103) to turn on Y120. Y120 latches itself on through the parallel branch. Press PB2 (X102) to break the latch and turn off Y120.

**I/O Mapping:**
- PB3 → X101 (Normally Open)
- PB4 → X100 (Normally Open)
- LED → Y121
- Timer → SR1C
