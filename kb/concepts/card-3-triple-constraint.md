# Card 3 Triple Constraint

- ### Card 3: Triple Constraint
**Q:** What are the 3 constraints in project management?

**A:**
1. **Time** (schedule/duration)
2. **Cost** (budget)
3. **Scope** (features/deliverables)

These must be balanced to maintain quality.

---

### Card 4: Duration Decreased ⚠️ QUIZ ANSWER
**Q:** What happens when the duration of the project is **decreased**?

**A:** 
- **Increase the cost** (hire more people/resources), AND/OR
- **Decrease the scope** (reduce features)

---

### Card 5: Cost Decreased ⚠️ QUIZ ANSWER
**Q:** What happens if the budget (cost) of the project is **decreased**?

**A:**
- **Increase the project duration** (reduce manpower), AND/OR
- **Decrease the scope** (reduce features)

---

### Card 6: Scope Increased ⚠️ QUIZ ANSWER
**Q:** What happens if project scope is **increased**?

**A:**
- **Increase project cost** (more resources needed), AND/OR
- **Increase project duration** (more time needed)

---

### Card 7: Project Management Goals
**Q:** What are the 3 main goals of project management?

**A:**
1. Complete project **on time**
2. Finish **within budget**
3. Achieve **customer satisfaction**

---

### Card 8: Project Phases
**Q:** What are the 5 phases in the full cycle of a project?

**A:**
1. Project initiation and organising
2. Project planning
3. Project execution
4. Project control and monitoring
