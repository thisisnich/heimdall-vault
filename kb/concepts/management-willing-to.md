# Management Willing To

- * — management willing to exceed original goal if necessary

### List of Major Deliverables (MD)

Defines what the project **is expected to produce**. Provides scope and boundaries for the project team and reduces rework cycle.

*Example — Setting up a New Restaurant:*

| MD1 (Furniture) | MD2 (Equipment) | MD3 (Dining Area) |
|---|---|---|
| Furniture sourced | Equipment sourced | Layout completed |
| Furniture purchased | Equipment purchased | Contractor sourced |
| Furniture delivered | Equipment delivered | Work completed |
| | Equipment installed | Furniture installed |

### Project Charter

A **document that formally authorizes a project**. Provides the PM with authority to apply organisational resources to project activities. Contains:

- Project title and description
- Project manager assigned and authority level
- Project objective statement
- Business case (why is the project done?)
- Product description/deliverables

**Validation & Concurrence required from:**
- **Team** — approve project definition, document all assumptions, provide clear understanding of project description and priorities
- **Management** — present project description, objective statement, major deliverables, and flexible matrix for approval to proceed
