# Approach Q What Is The

- Approach
**Q:** What is the hybrid approach for IoT projects?

**A:** Many IoT projects use **PERT for R&D phases** and **CPM for production phases**:
- **Design/Prototype phase:** PERT (high uncertainty)
- **Development phase:** PERT for new tech, CPM for standard tasks
- **Testing phase:** CPM (well-defined test procedures)
- **Production/Deployment:** CPM (standardized processes)

---

## 🎯 Quick Review - Must-Know Answers

### Triple Constraint Quiz Answers (MEMORIZE!)
1. **Duration decreased** → Increase cost AND/OR decrease scope
2. **Cost decreased** → Increase duration AND/OR decrease scope
3. **Scope increased** → Increase cost AND/OR increase duration

### Key Numbers to Remember
- Core team: **Max 7 members**
- Project objective statement: **Max 25 words, 1 target**
- WBS major components: **4 to 7**
- WBS minimum tasks per breakdown: **2 tasks**
- Status reporting frequency: **At least once a month**

### Key Roles
- **Sponsor:** Approves, sets direction, resolves conflicts
- **PM:** Plans, executes, monitors, reports
- **Core Team:** Makes major decisions (max 7)
- **Extended Team:** Provides expertise (no limit)

### WBS Key Points
- Sequence: **NOT important**
- Completeness: **IS important**
- Naming: **Verb/noun**
- Approach: **Top-down** (validate bottom-up)
- Process: **Team effort** (not PM alone)
