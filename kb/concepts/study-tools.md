# Study Tools

- # 📚 Study Tools - Centralized Flashcards & MCQ Quizzes

**Interactive study tools with Notion-inspired design that work directly in Obsidian!**

---

## 🎯 What's Included

### **🆕 Universal Apps (Recommended)**

#### 1. **Universal Flashcards** (`universal-flashcards.html`) ⭐
- **ONE app for ALL flashcard decks**
- Beautiful Notion-inspired warm minimalist design
- Dropdown selector to choose any deck
- Smart filters (All, Critical, By Chapter)
- Progress tracking and shuffle feature
- Keyboard shortcuts (←/→ arrows, Space to flip)

#### 2. **Universal Quiz** (`universal-quiz.html`) ⭐
- **ONE app for ALL MCQ quizzes**
- Notion-style clean interface
- Choose quiz and topic from dropdowns
- Instant feedback with explanations
- Score tracking, timer, and grade calculation
- Keyboard shortcuts (1-4 for answers, Enter for next)

---

## 📁 Centralized Deck System

### **Flashcard Decks** (`flashcard-decks/` folder)
All flashcard content stored in separate `.js` files:
- `ege322-test1.js` - EGE322 IoT Project Management (36 cards)
- `ap-cs-basics.js` - AP CS A Java Basics (10 cards)
- `ap-calc-derivatives.js` - AP Calc BC Derivatives (10 cards)
- `ap-calc-integrals.js` - AP Calc BC Integration (10 cards)

### **Quiz Decks** (`quiz-decks/` folder)
All quiz content stored in separate `.js` files:
- `ege322-test1.js` - EGE322 Chapter 2.1 & 2.2 quizzes
- `ap-cs-basics.js` - AP CS A Java Fundamentals
- `ap-calc-bc.js` - AP Calc BC Derivatives & Integration
