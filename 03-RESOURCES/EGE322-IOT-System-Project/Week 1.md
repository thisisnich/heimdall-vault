#### IoT System Project [60 hours]

This module integrates knowledge and skills in wireless and embedded technologies to implement Internet of Things (IoT) applications. The module also covers basics of project management, in particular, issues related to the organization, planning, realization and control of projects. Learners will be able to conceptualise and develop solutions to solve demand-driven problem statements.




Synopsis

This module integrates knowledge and skills in wireless and embedded technologies to implement Internet of Things (IoT) applications. The module also covers basics of project management, in particular, issues related to the organization, planning, realization and control of projects. Learners will be able to conceptualise and develop solutions to solve demand-driven problem statements.

Module Learning Outcome
