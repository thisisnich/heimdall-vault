# Week 1: Network Fundamentals

## 🌐 Introduction to Networks

### Network Purpose
Networks are essential to connect two devices to each other for communication and resource sharing.

### Data Communications
Data communications refers to data communication between two devices through a transmission medium.

---

## 📡 Transmission Systems

### Connection Types
- **Wired:** Fiber optic cables
- **Wireless:** WiFi, 3G, 4G LTE, 5G

### Signal Evolution
- **Historical:** Data sent and received using analog waveforms
- **Modern:** Data typically sent using digital signals (1s and 0s)
- **Transition:** The world constantly moves between analog and digital
- **Conversion:** Coders handle analog to digital conversion and vice versa

---

## ⚡ Data Rate

### Definition
- **Also called:** Bit rate
- **Measurement:** Number of bits transmitted per second

### Importance
- Determines transmission speed
- Affects network performance
- Critical for application requirements

---

## 🔗 Key Concepts

### Network Components
- **Devices:** Endpoints that communicate
- **Medium:** Transmission path for data
- **Protocols:** Rules for communication

### Communication Process
1. **Data Generation:** Creating information to transmit
2. **Encoding:** Converting to transmittable format
3. **Transmission:** Sending through medium
4. **Reception:** Capturing at destination
5. **Decoding:** Converting back to usable format
