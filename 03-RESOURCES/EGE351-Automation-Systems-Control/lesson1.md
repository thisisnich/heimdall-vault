# EGE351 - Automation Systems & Control

> **Related:** [[Lecture_1_PLC|Lecture 1: PLC]] | [[EGE351 Automatino Systems & Control/lab1|Lab 1 Notes]]

---

## Synopsis

Automation systems are crucial to many industries that desire to increase productivity, improve product quality and streamline processes. Through this module, learners will be equipped with knowledge and skills about various fieldbus systems, application and interface of sensors and actuators, programming of automation controller and implementation of Internet of Things (IoT) in automation systems. They will also be able to develop, integrate and troubleshoot industrial automation systems.

> **IoT Context:** See [[ch0#Embedded System in Internet of Things (IoT)|IoT in Embedded Systems]]

---

## Learning Outcomes

At the end of this module, learners will be able to:

- Build various automation applications by using appropriate controllers, sensors and actuators
- Evaluate PLC-based and microcontroller-based automation controllers in terms of their architecture and applications → See [[Lecture_1_PLC#PLC vs Other Control Systems|PLC Comparison]]
- Interface peripheral devices to automation controller according to application requirements
- Build software algorithms to meet application requirements using industry-standard automation controller platform
